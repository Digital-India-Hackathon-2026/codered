# Font Assets

Place the following font files in this directory, then run:
  npx react-native-asset

## Fraunces (Serif — Display/Headings)
- Fraunces-Regular.ttf
- Fraunces-Medium.ttf
- Fraunces-SemiBold.ttf
- Fraunces-Bold.ttf

## GeneralSans (Sans — UI/Body)
- GeneralSans-Regular.ttf
- GeneralSans-Medium.ttf
- GeneralSans-Semibold.ttf
- GeneralSans-Bold.ttf

Download Fraunces from Google Fonts: https://fonts.google.com/specimen/Fraunces
Download GeneralSans from Fontshare: https://www.fontshare.com/fonts/general-sans
